# Kerubi

Kerubi is a minimalist task manager designed for focused productivity. This is an early development version (v0.1) focused on core functionality and dynamic customization.

## Features

* **Task Dual-View**: Separate sections for today's priorities and upcoming tasks.
* **Smart Status**: Tasks automatically mark themselves as expired once the deadline passes.
* **Step System**: Support for creating simple reminders or complex step-by-step series.
* **Dynamic Themes**: Over 20 built-in visual environments with instant switching.
* **Local Storage**: Data persists locally in your browser without the need for a database.

## Current Project Status

This project is currently in its **initial development phase**. Current goals include:
* Refining the user interface and transitions.
* Implementing the core task expiration logic.
* Expanding the theme engine.

## Usage

1. Open `index.html` in any web browser.
2. Click the action button to create a new task.
3. Manage themes via the navigation bar.

## Technology Stack
